# Aplicativo de Gerenciamento de Pastas e Arquivos

## Visão Geral

Desenvolvi um aplicativo utilizando Next.js, Tailwind CSS e TypeScript, focado na organização e gerenciamento de pastas e 
arquivos de forma eficiente e intuitiva. O armazenamento é realizado através do sistema de arquivos (FS) do JavaScript.
O app possui um crud, criando, renomeando, apagando arquivos e pastas.

## Funcionalidades Principais

### Criação de Pastas e Arquivos

O aplicativo permite a criação de pastas, onde cada pasta é identificada por um nome central. Dentro de cada pasta, os submenus são arquivos individuais, cada um com um editor embutido.

### Editor de Texto (EditorJS)

Cada arquivo dentro do aplicativo possui um editor de texto avançado que oferece várias ferramentas para a criação de conteúdo. As funcionalidades do editor incluem:

##### Lista ferramentas : 
- Header
- LinkTool
- Embed 
- Table 
- List 
- Warning
- CodeTool 
- SimpleImage 
- RawTool 
- Quote 
- Marker 
- Checklist 
- Delimiter 
- InlineCode 
- AlignmentTuneTool 
- Paragraph raph
- AIText 
- NestedList

### Interface de Usuário

Com o uso do Tailwind CSS, a interface do usuário é responsiva e moderna, garantindo uma experiência agradável e intuitiva. 
A combinação de Next.js e TypeScript proporciona um desempenho otimizado e uma base de código bem estruturada.


## Conclusão

Este aplicativo é uma ferramenta para a organização e edição de arquivos e pastas com uma interface amigável e 
funcionalidades avançadas. Ideal para usuários que buscam eficiência e praticidade no gerenciamento de seus documentos digitais.
