# O Que Foi Aprendido

## Desenvolvimento com Next.js, Tailwind CSS e TypeScript

Ao desenvolver um aplicativo de gerenciamento de pastas e arquivos, aprendi a utilizar Next.js, 
Tailwind CSS e TypeScript de maneira integrada. Aprendi a utilizar páginas dinâmicas no next, como utilizar 
hooks do react como useState, useEffect, useContext.
Aprendi a utilizar temas, layouts e como funciona o sistema de rotas do nextjs.

### TypeScript
Aprendi a utilizar as funcionalidades básicas e algumas intermediarias do TypeScript, tipando uma variedade de itens.

## Criação de Funcionalidades

### Organização de Pastas e Arquivos

Aprendi a implementar um sistema de criação e gerenciamento de pastas e arquivos, onde cada pasta possui um nome 
central e os submenus são arquivos individuais. Essa estrutura permite uma organização clara e intuitiva dos documentos.

### Editor de Texto

Utilizei o EditorJS e aprendi a integrar ele ao meu aplicativo, configurando várias ferramentas que ele possui.

## ShadCN UI
Utilizei esta Biblioteca que ajuda na criação do design do app, e que me auxiliou muito pois e muito fácil e simples de 
utilizar para a crição de vários componentes.

## Armazenamento com o Sistema de Arquivos (FS) do JavaScript

Utilizei FS do javascript para salvar os dados em arquivos, em que ficam a configuração de todo o menu e do editor, assim
ao criar o menu é utilizado um loop que percorre todas as pastas e arquivos e monta o menu, também adicionando conteúdo ao 
editor e salvando. 

## Design de Interface de Usuário

Com o uso do Tailwind CSS, aprendi a estilizar o app, deixando ele mais amigavél e bonito e bem simplista.

## Dificuldades
A parte em que estive com maior dificuldade foi para a criação do menu dinâmico pois precisei criar funções que utilizaram
o FS (Módulo de sistema de arquivos do node ), a configuração do EditorJS também estive com algumas dificuldades pois a 
documentação para alguns componentes não estão completas.


## Conclusão

O desenvolvimento deste aplicativo me proporcionou um aprendizado valioso em diversas áreas do desenvolvimento web, desde a 
utilização de tecnologias modernas como Next.js, Tailwind CSS e TypeScript, até a implementação de funcionalidades avançadas de 
edição e gerenciamento de arquivos. Essa experiência consolidou meus conhecimentos e habilidades, preparando-me para enfrentar novos 
desafios no desenvolvimento de aplicações web.
